# vue-shop-car

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
### 获取本地json数据
```
this.$ajax.get('http://localhost:8080/shop.json').then((res) => {})
```
```
shop.json放public文件夹下
```
